# Prompt Manager (ComfyUI)

Локальный менеджер positive prompt для ComfyUI. Сервер запускается на Windows, UI открывается с Mac через LAN.

## Установка (Windows)
1. Установите Node.js 18+.
2. В корне проекта:
   ```bash
   npm install
   ```
3. Режим разработки:
   ```bash
   npm run dev
   ```
   - сервер: http://localhost:8787
   - клиент: http://localhost:5173

4. Продакшн сборка:
   ```bash
   npm run build
   npm run start
   ```
   Сервер раздаёт собранный клиент по адресу `http://<WINDOWS_IP>:8787`.

## Доступ с Mac
- Узнайте IP Windows в локальной сети.
- Откройте в браузере: `http://WINDOWS_IP:8787`.

## Экспорт prompt
- По умолчанию запись идёт в `C:\CU\text_input\prompt.txt`.
- Путь можно изменить в разделе **Настройки**.
- Кнопка **Export** перезаписывает файл и сохраняет запись в истории.

## Проверка пути
В **Настройки** → **Проверить путь** можно убедиться, что сервер имеет доступ к файлу. Директория создаётся автоматически.

## LM Studio
В **Настройки** заполните Base URL / API Key / Model. Панель перевода находится в разделе **Фразы**.

## Тесты (backend)
```bash
npm run test
```

## Резервная копия
В **Настройки** доступен экспорт/импорт базы в JSON (backup).
